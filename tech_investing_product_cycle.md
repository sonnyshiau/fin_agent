# Tech Investing: Product Cycle 與 Catalyst 思維

這段筆記整理一個核心觀念：優秀的 tech investor 看的不是單一新聞，而是 product cycle、供應鏈變化，以及未來 2-18 個月可能出現的催化劑。

## 1. Tech Investors 主要在玩 Product Cycle

Tech investors 要永遠記得，科技投資的核心往往不是追逐已經公開的訊息，而是在研究 product cycle。

投資人最喜歡的機會，是新科技帶來供應鏈大洗牌。例如一代新平台或新架構推出時，會重新分配價值鏈，也會讓某些環節突然變得稀缺、重要，甚至出現新的贏家。

以 Vera Rubin 以及相對應的 AI hardware 供應鏈變化為例，真正重要的問題包括：

- 哪些零組件或環節的 content value 變高？
- 哪些地方的供需狀況最緊張？
- 哪些廠商拿到 design win？
- 什麼時候開始放量？
- 下一代平台的設計改變，會對哪些 component 有正面或負面影響？

這些問題不是等法說會、公告或新聞出來才開始研究。等到市場大眾看到這些資訊時，通常已經是舊訊息。

## 2. Channel Check 是提前取得能見度的關鍵

如果有認真做 channel check，投資人理論上可以比一般市場早 6-9 個月掌握供應鏈變化。

更認真的投資人，甚至已經開始研究下一代平台。例如在市場還聚焦 Vera Rubin 時，領先的投資人可能已經在研究 Feynman 的設計差異，以及這些差異對各類 component 的影響。

換句話說，一個優秀的 tech investor 至少應該保持 12-18 個月的 product cycle 能見度。

這種能見度不代表每個細節都會完全正確。大方向可以先建立，但中間一定會有規格、時程、供應商分配或需求強度的更改與調整。因此，持續做 channel check，是維持掌握度的必要工作。

## 3. 短期交易要看 Catalyst

如果問題是短期要炒什麼，就要從 catalyst 的角度思考。

高頻報價就是一種催化劑。當市場不斷看到價格上漲、交期拉長、供給緊張或訂單動能變強，股價就容易被重新定價。

如果沒有明確報價可以追，就要反過來想：

> 2-3 個月後，新聞可能會寫什麼？

這是另一套短期推演方法。重點不是等待新聞發生，而是提前想像市場接下來會被什麼敘事、數據或事件喚醒。

## 4. 投資研究的實務含義

在 tech investing 裡，研究的速度與深度很重要。真正有價值的研究，應該努力回答以下問題：

- 下一個 product cycle 是什麼？
- 新平台或新技術改變了哪些供應鏈價值分配？
- 哪些 component 的 content value 上升？
- 哪些環節可能最缺？
- 哪些公司已經取得 design win？
- 放量時間點落在哪裡？
- 市場現在還沒有反映的是什麼？
- 2-3 個月後，可能出現什麼 catalyst？

如果等到公司法說會、公告或主流新聞才開始反應，通常已經太慢了。
