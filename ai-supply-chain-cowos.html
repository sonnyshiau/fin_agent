MS / UBS 對照

核心結論：MS 與 UBS 都指向 2027 CoWoS 仍偏吃緊，但敘事重點不同。MS 對總需求、TSMC 產能與非 TSMC 產能貢獻較樂觀；UBS 更強調 ASE / OSAT full-process CoWoS 能力打開後段封裝市場。

GPU / ASIC HBM CoWoS OSAT ABF

共同點

Nvidia Rubin / Vera 是 2027 主軸，Google ASIC / TPU 讓 CoWoS 需求不只靠 GPU，TSMC 高階封裝仍是最高價值瓶頸。

最大分歧

不是 Nvidia，而是 AMD Venice、Amkor / 非 TSMC 產能，以及 ASIC 客戶把封裝分流到 OSAT 的速度。

投資含義

若 MS 成真，上行彈性在 AMD、Amkor、ABF / HBM pull-through；若 UBS 成真，較乾淨的受益主線是 TSMC、ASE、MediaTek。

追蹤訊號

TSMC 後段封裝營收、ASE 資本支出 / 稼動率、AMD Venice ramp、Google TPU 分配、HBM 交期與 ABF 供給。

MS 隱含的情境

更高的 TSMC 產能、非 TSMC 產能、AMD Venice booking 與 ASIC 需求轉出貨。這是一個更積極的供應鏈分配情境。

UBS 隱含的情境

TSMC / ASE 擴產速度本身就是 cloud AI 需求的領先指標。UBS 的優點是較容易用營收、資本支出、稼動率追蹤。

市場可能重估的位置

若 2-3 個月後新聞開始寫 AMD Venice 提前拉貨、Google TPU 訂單上修、ASE CoWoS 稼動率，相關供應鏈可能提前反應。

會破壞 thesis 的因素

HBM / ABF supply 無法跟上、Rubin / Venice timing 變慢、ASIC 客戶轉單不如預期，或 OSAT full-process yield ramp 慢於報告假設。

第 4 頁 / 技術與受益者

技術路徑決定誰吃到瓶頸價值

CoWoS-S/L/R、FoCoS、EMIB-T、CoPoS、HBM 不是同一層東西。投資上要分清楚：哪些是 2027 主軸，哪些是中期替代或擴產路徑。

tech map

CoWoS / 封裝技術矩陣

把報告中提到的 key 技術整理成一張投資用途表。

技術

CoWoS-S 2027 主軸

Silicon interposer 路線，Google TPU / ASIC 常見。投資重點是 TSMC 高階封裝產能與 interposer 供給。

CoWoS-L 2027 主軸

Local silicon interconnect / larger package route，Nvidia Rubin、AMD MI / Venice exposure 高。瓶頸更直接映射到 GPU / CPU ramp。

CoWoS-R 選擇性採用

報告中連到部分 Marvell / Trainium 分配。重點是 ASIC 客戶是否把更多封裝分流到 OSAT。

FoCoS OSAT

ASE/SPIL 的非 TSMC 高階封裝能力之一。若 UBS 的 ASE 打開市場空間情境成真，FoCoS / full-process 能見度上升。

EMIB-T 中期延伸

Intel 高階封裝路線，MS 提到 Google TPU v9 相關可能性。較像中期產能延伸，不是 2027 CoWoS 即刻替代。

CoPoS 2028+

Panel-level packaging 延伸。公開資料偏向 2028-29 ramp；投資上是中期 throughput story。

HBM 含量提升

MS 估 2027 HBM 需求 51bn Gb。CoWoS 需求越高，HBM 堆疊、測試、基板拉貨越強。

ABF 供給限制

報告暗示 ABF supply 會影響 MediaTek TPU 27/28 shipments。這是 2-3 個月新聞流可能重新定價的追蹤點。

ASIC 需求來源

Google、AWS、Meta ASIC 讓 CoWoS 需求不只看 Nvidia。UBS 的 ASIC 33% share 是重要分散訊號。

受益者 / 追蹤訊號

受益層級與下一步該看什麼。

追蹤清單

直接產能受益者

TSMC 吃到最高價值瓶頸。ASE 受益於 full-process CoWoS 擴張。Amkor 是 MS non-TSMC 產能假設中的 swing factor。

平台 / ASIC 拉動

AMD 是兩家券商最大分歧。MediaTek 與 Broadcom 受益於 Google TPU。Alchip / Marvell / GUC 則取決於 ASIC design slot。

二階供應鏈

HBM、advanced substrate、ABF 與 testing 產能需要一起追蹤；瓶頸可能從封裝產能轉移到零組件供給。
