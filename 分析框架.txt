│   ├── 1.1 一句話判斷

│   │   ├── 上游：材料 / 設備 / IP / 零組件

│   │   ├── 下游：雲端 / 伺服器 / AI 平台 / 終端應用
│   │   └── 判斷：公司是否站在產業瓶頸位置？

│   ├── 1.3 商業模式
│   │   ├── 賣晶片
│   │   ├── 賣模組

│   │
│   └── 1.4 核心矛盾
│       ├── 這家公司為什麼會漲？
│       ├── 市場現在相信它什麼？
│       ├── 市場最擔心它什麼？
│       └── 核心矛盾不是「公司好不好」，而是「預期是否已經太滿」
│
├── 二、預期管理階段
│   ├── 2.1 公司目前處在哪個階段？
│   │   ├── 敘事期：市場買的是故事與 TAM
│   │   ├── 業績驗證期：市場開始看收入、客戶、訂單
│   │   ├── 強勢兌現期：財報全面超預期
│   │   ├── 完美定價期：好消息大多已反映在股價
│   │   └── 預期反轉期：成長放緩或指引轉弱，估值開始收縮
│   │
│   ├── 2.2 預期管理判斷
│   │   ├── 公司是否連續上修指引？
│   │   ├── 市場是否已經把未來 1–2 年成長提前定價？
│   │   ├── 股價反應是「漲財報」還是「漲預期」？
│   │   ├── 財報好但股價跌，代表預期可能已太高
│   │   └── 財報普通但股價漲，代表市場可能看見下一階段成長
│   │
│   └── 2.3 結論
│       ├── 早期敘事期：可以用 TAM 與產品路線估值
│       ├── 業績驗證期：重點看 revenue growth
│       ├── 強勢兌現期：重點看 beat and raise
│       ├── 完美定價期：重點看風險報酬比
│       └── 反轉期：重點看估值殺傷與下修幅度

├── 三、關鍵財務指標
│   ├── 3.1 收入增長
│   │   ├── 最新季度收入

│   │   ├── QoQ 環比增長
│   │   ├── 下一季指引
│   │   ├── 全年收入預期
│   │   └── 判斷標準
│   │       ├── 優秀：QoQ > 15%，YoY > 50% 或明顯高於同業
│   │       ├── 合格：QoQ 8%–15%，YoY 穩定高成長
│   │       └── 危險：QoQ < 5%，或指引不再上修

│   │   ├── 週期股：中段 EPS / EBITDA
│   │   └── 平台型公司：收入 CAGR + 長期 margin structure
│   │
│   ├── 8.3 Forward PE 是否適用？
│   │   ├── 適用條件
│   │   │   ├── 公司已經穩定獲利
│   │   │   ├── EPS 具備可預測性
│   │   │   ├── 營運槓桿已經出現
│   │   │   ├── 毛利率和營業利益率相對穩定
│   │   │   └── 市場開始用 earnings power 定價
│   │   ├── 不適用條件
│   │   │   ├── 公司仍在燒錢
│   │   │   ├── EPS 波動極大
│   │   │   ├── SBC 或一次性費用扭曲 EPS
│   │   │   ├── 利潤率尚未穩定
│   │   │   └── 市場主要定價 TAM，而非當期獲利
│   │   └── 結論
│   │       ├── 可用：主估值 / 輔助估值 / 不適合
│   │       └── 高成長 AI 半導體股通常只能把 Forward PE 當輔助
│   │
│   ├── 8.4 PEG 是否適用？
│   │   ├── PEG 公式
│   │   │   └── PEG = Forward PE / 未來 EPS CAGR
│   │   ├── 適用條件
│   │   │   ├── EPS 成長率可合理預測
│   │   │   ├── 公司已穩定獲利
│   │   │   ├── 非一次性爆發
│   │   │   ├── margin 結構穩定
│   │   │   └── 未來 2–3 年 EPS CAGR 有可信度
│   │   ├── 不適用條件
│   │   │   ├── EPS 基期太低
│   │   │   ├── 剛轉盈
│   │   │   ├── 單一產品或單一客戶導致成長失真
│   │   │   ├── 半導體週期造成 EPS 劇烈波動
│   │   │   └── 成長高度依賴市場情緒
│   │   └── 結論
│   │       ├── 成熟成長股：PEG 有參考價值
│   │       ├── 高成長早期股：PEG 容易失真
│   │       └── AI 半導體高成長股：只能輔助，不可單獨使用
│   │

│       ├── 若成長放緩，multiple contraction 空間多大？
│       └── 目前風險報酬比：好 / 普通 / 差

│   ├── 9.1 Bull Case：樂觀情境

│   │   ├── 股價驅動

│   │   │   ├── Multiple 維持高位或擴張

│   │   └── 投資策略
│   │       ├── 持有核心倉位

│   │       └── 不因估值高而過早賣出

│   │   │   └── 新產品逐步放量但不爆炸式超預期
│   │   ├── 財務表現
│   │   │   ├── 收入符合或小幅優於指引
│   │   │   ├── EPS 穩定成長
│   │   │   ├── 營業利益率維持健康
│   │   │   └── FCF 持續改善
│   │   ├── 股價驅動
│   │   │   ├── 股價跟隨 earnings growth
│   │   │   ├── Multiple 不再大幅擴張
│   │   │   └── 報酬主要來自基本面兌現
│   │   └── 投資策略
│   │       ├── 合理估值內分批持有
│   │       ├── 高估時不追價
